# Chapter 1 - Mastering Stack-Based Computing with Forth

## What is Forth?

* Part of the GNU ecosystem
* Implementation is known as **GNU Forth** or `gforth`

## Installation

* Installed as `gforth`
* Best and earliest version to install is `gforth-0.7.9`
  * Earlier versions fail to compile with examples from the book
* Link in Book: [gforth](https://www.gnu.org/software/gforth/)
  * Very slow
  * Leads to page w/ basic information about `gforth`
* Link to Savannah git Repo: [gforth.git](https://cgit.git.savannah.gnu.org/cgit/gforth.git/)
  * Clone repo: `git clone git://git.git.savannah.gnu.org/gforth.git`
